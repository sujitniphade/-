# 🙏 Datt Jap Counter

A simple Python GUI application built with **Tkinter** for counting **Dattatreya Mantra Japa**.

## Features

* 🧘 Counts mantra chants up to **1111**
* 🎨 Highlights each mantra word one by one
* ⏱ Adjustable counter interval
* ✨ Adjustable highlight interval
* 📊 Progress bar to track completion
* ▶ Start, ⏸ Pause, and 🔄 Reset controls
* 🇮🇳 Supports Devanagari mantra display

## Mantra

```
ॐ द्रां दत्तात्रेयाय नमः
```

## Requirements

* Python 3.x
* Tkinter (included with most Python installations)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/datt-jap-counter.git
```

2. Open the project folder:

```bash
cd datt-jap-counter
```

3. Run the application:

```bash
python datt_jap_counter.py
```

## How to Use

1. Set the **Counter Time** (seconds).
2. Set the **Highlight Time** (seconds).
3. Click **Start** to begin chanting.
4. Click **Pause** to stop temporarily.
5. Click **Reset** to start over.
